# COOKINcat
El juego sobre un gato hambriento y desempleado que consigue trabajo como cocinero en casa de una bruja. No sabe cocinar (ni leer) pero con la ayuda de un libro mágico (Libro de Chicookte) mantiene su puesto, plato a plato. Deberá aprender a gestionar la despensa, el apetito de la bruja y a hacer platos de rechupete.
